# Простое CRUD приложение на Vue3 и FastAPI (Python ≥ 3.10)
#### --- not build version

### Для запуска backend части:
1) cd backend
2) python -m venv venv
3) venv\Scripts\Activate.ps1
4) pip install -r req
5) uvicorn mian:app --port 8000

### Для запуска frontend части (в отдельном терминале):
1) cd frontend
2) npm install
3) npm run dev
